# AI提示词文档

## 1. 概述

本文档提供了一系列AI提示词模板，用于快速生成符合项目规范的代码。

## 2. 代码生成提示词

### 2.1 生成实体类

```
请为我生成一个实体类，要求如下：
- 类名：{类名}
- 功能：{功能描述}
- 字段：{字段列表}
- 技术栈：Spring Boot + JPA + Lombok
- 规范：
  1. 使用@Data注解
  2. 使用@Entity和@Table注解
  3. 主键使用@GeneratedValue(strategy = GenerationType.IDENTITY)
  4. 添加@Version乐观锁字段
  5. 添加createdAt和updatedAt字段
  6. 使用@PrePersist和@PreUpdate自动填充时间
  7. 字段添加@Column注解，设置nullable和length
```

### 2.2 生成Mapper接口

```
请为我生成一个Mapper接口，要求如下：
- 接口名：{接口名}
- 实体类：{实体类名}
- 功能：{功能描述}
- 技术栈：Spring Data JPA
- 规范：
  1. 继承JpaRepository
  2. 使用@Repository注解
  3. 自定义查询使用@Query注解
  4. 更新操作使用@Modifying注解
  5. 参数使用@Param注解
```

### 2.3 生成Service类

```
请为我生成一个Service类，要求如下：
- 类名：{类名}
- 功能：{功能描述}
- 依赖：{依赖的Mapper和其他Service}
- 技术栈：Spring Boot
- 规范：
  1. 使用@Service和@Slf4j注解
  2. 使用@RequiredArgsConstructor构造注入
  3. 业务方法添加@Transactional注解
  4. 使用log记录关键日志
  5. 异常使用BusinessException
  6. 添加完整的JavaDoc注释
```

### 2.4 生成Controller类

```
请为我生成一个Controller类，要求如下：
- 类名：{类名}
- 功能：{功能描述}
- 基础路径：{路径}
- 依赖：{依赖的Service}
- 技术栈：Spring Boot + Swagger
- 规范：
  1. 使用@RestController和@RequestMapping注解
  2. 使用@Tag和@Operation注解
  3. 使用@RequiredArgsConstructor构造注入
  4. 返回值使用ApiResult封装
  5. 参数校验使用@Valid注解
  6. 限流使用@RateLimit注解
```

### 2.5 生成分布式锁实现

```
请为我生成一个分布式锁实现，要求如下：
- 类名：{类名}
- 功能：{功能描述}
- 锁类型：Redis分布式锁
- 技术栈：Spring Boot + Redis
- 规范：
  1. 使用StringRedisTemplate
  2. 使用SET NX EX命令获取锁
  3. 使用Lua脚本释放锁
  4. 支持锁续期
  5. 支持重试机制
  6. 记录详细日志
```

### 2.6 生成限流实现

```
请为我生成一个限流实现，要求如下：
- 类名：{类名}
- 功能：{功能描述}
- 限流算法：{滑动窗口/固定窗口/令牌桶}
- 技术栈：Spring Boot + Redis
- 规范：
  1. 使用Lua脚本保证原子性
  2. 支持分布式环境
  3. 支持多种限流策略
  4. 记录限流日志
  5. 异常时放行，保证可用性
```

### 2.7 生成幂等注解

```
请为我生成一个幂等注解和切面，要求如下：
- 注解名：{注解名}
- 功能：防重复提交
- 技术栈：Spring Boot + AOP + Redis
- 规范：
  1. 注解支持SpEL表达式
  2. 支持自定义过期时间
  3. 支持自定义错误提示
  4. 使用Redis存储幂等键
  5. 业务失败后删除幂等键
```

## 3. 问题排查提示词

### 3.1 分析异常日志

```
请帮我分析以下异常日志，找出问题原因并提供解决方案：

异常信息：{异常堆栈}

上下文：
- 操作：{执行的操作}
- 参数：{传入的参数}
- 环境：{运行环境}
```

### 3.2 性能优化建议

```
请帮我分析以下代码的性能问题，并提供优化建议：

代码功能：{功能描述}
当前性能：{性能指标}
目标性能：{目标指标}

代码：
{代码内容}
```

### 3.3 安全漏洞检查

```
请帮我检查以下代码是否存在安全漏洞：

代码功能：{功能描述}
代码：
{代码内容}

请检查以下方面：
1. SQL注入
2. XSS攻击
3. CSRF攻击
4. 敏感信息泄露
5. 权限控制
```

## 4. 文档生成提示词

### 4.1 生成API文档

```

功能描述：{功能描述}

请包含：
1. 接口概述
2. 请求参数说明
3. 响应参数说明
4. 请求示例
5. 响应示例
6. 错误码说明
```
